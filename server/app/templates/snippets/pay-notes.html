Additional Notes
  • In order to test Masterpass, you will need to set up a test account with Mastercard. On the Checkout form, click on the Masterpass logo and follow the instructions. You can use any test card number.
  • In order to test Visa Checkout, you will need to set up a test account with Visa. On the Checkout form, click on the Visa Checkout logo and follow the instructions. You can use any test card number.
  • For testing Interac Online, you can use any bank account information.
